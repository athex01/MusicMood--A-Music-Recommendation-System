Movie Recommendation System

Setup Instructions:

Step 1: Project Setup
---------------------
1. Create a folder in your desired location.
2. Paste the following files into that folder:
   - application.py
   - Data.csv
   - recommendation_engine.py
   - Target.csv

Step 2: Open in Code Editor
---------------------------
3. Open the folder in VS Code or any other source code editor of your choice.

Step 3: Install Python
----------------------
4. Make sure Python is installed on your system.
   - If not installed, download it from https://www.python.org/ and install.
   - Verify installation by running:
     python --version

Step 4: Run the Application
---------------------------
5. In your terminal or command prompt, navigate to the project folder and run:
   python application.py

Step 5: Enjoy
-------------
6. Give input as prompted in the terminal and explore movie recommendations.
7. Have fun using your personalized recommendation system!

Files Description:
------------------
application.py          - Main application file to run.
recommendation_engine.py - Contains recommendation logic.
Data.csv                - Dataset for recommendations.
Target.csv              - Target data for user reference.

Notes:
------
- Ensure all files are in the same folder before running.
- You can modify the datasets to include more movies as per your use case.
